# PASERK Type: lid

This kind of PASERK serves as a pointer to another PASERK, which is in
turn intended for `local` PASETOs.

## Format

    k[version].lid.[data]

## Operations

See [ID](../operations/ID.md) for the algorithm used to derive local IDs.
