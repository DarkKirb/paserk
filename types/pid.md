# PASERK Type: pid

This kind of PASERK serves as a pointer to another PASERK, which is in
turn intended for `public` PASETOs.

This is for serializing **public keys** in a keypair.

## Format

    k[version].pid.[data]

## Calculations

See [ID](../operations/ID.md) for the algorithm.
